# Android 15-18 Upgrade (2026)

## Overview
Upgraded FireKidsGame project to support Android 15-18 (API levels 34-36), targeting Android 15 (API 34) for testing.

## Changes Made

### 1. Gradle Build Tool Upgrade
**File:** `build.gradle` (root)
- **Android Gradle Plugin:** `4.2.2` → `8.3.2`
  - Required for Android API 34+ support
  - Provides compatibility with latest Android build tools and features

### 2. Gradle Wrapper Upgrade
**File:** `gradle/wrapper/gradle-wrapper.properties`
- **Gradle Version:** `6.7.1` → `8.4`
  - Required by Android Gradle Plugin 8.3.2
  - Provides better performance, security updates, and newer features

### 3. Android SDK Configuration
**File:** `android/build.gradle`
- **compileSdkVersion:** `32` → `35` (Android 16)
- **buildToolsVersion:** `32.0.0` → `35.0.0`
- **targetSdkVersion:** `32` → `35` (Android 16)
- **minSdkVersion:** `16` → `23` (Android 6.0 Marshmallow)
  - API 16-22 is deprecated and incompatible with modern Google Play Services
  - API 23 is the minimum for Google Mobile Ads SDK 25.0.0
- **versionCode:** `35` → `38`
- **versionName:** `"35"` → `"1.33"`
- **Added namespace:** `'com.lh9.feg1.firekidsgame'`
  - Required for AGP 8.0+, replaces package declaration in manifest
  - Namespace is now declared in build.gradle instead of AndroidManifest.xml

**In-app version display:**
- **Version string:** `"Version 1.32"` → `"Version 1.33"` (in Variables.java)

### 4. Java Version Upgrade
**File:** `android/build.gradle`
- **sourceCompatibility:** `JavaVersion.VERSION_1_8` → `JavaVersion.VERSION_11`
- **targetCompatibility:** `JavaVersion.VERSION_1_8` → `JavaVersion.VERSION_11`
  - Required for AGP 8.0+ and modern Android development
  - Java 11 is minimum for Android Gradle Plugin 8.x

### 5. AndroidX Migration
**File:** `gradle.properties`
- **Added:** `android.useAndroidX=true`
- **Added:** `android.enableJetifier=true`
  - Migrates legacy Android Support Library to AndroidX
  - Jetifier automatically converts legacy dependencies to AndroidX equivalents
  - Required for compatibility with modern Android libraries

### 6. Google Play Services Ads Update
**File:** `build.gradle` (root)
- **play-services-ads:** `20.6.0` → `25.0.0`
  - Latest stable version as of February 2026
  - No known vulnerabilities
  - Includes security updates, performance improvements, and new ad features
  - Versions 23.x are deprecated as of Feb 2026
  - Maintains full backward compatibility with existing ad implementation code
  - Requires minimum API 23 (Android 6.0)

### 7. Android Manifest Update
**File:** `android/AndroidManifest.xml`
- **Removed:** `package="com.lh9.feg1.firekidsgame"` attribute from manifest tag
  - Package declaration moved to namespace in android/build.gradle (AGP 8.0+ requirement)
  - Manifest no longer declares package; build system handles it
- **Added:** 16KB page size support property
  ```xml
  <property
      android:name="android.app.property.16KB_PAGE_SIZE_SUPPORT"
      android:value="true" />
  ```
  - Required for Android 15+ devices with 16KB memory pages
  - Enables compatibility with newer ARM64 devices (Pixel 9, etc.)
  - Google Play Store requirement for apps targeting API 35+
  - Ensures app works on devices with both 4KB and 16KB page sizes

### 8. Dependency Conflict Resolution
**File:** `android/build.gradle`
- **Removed:** `implementation 'org.jetbrains:annotations-java5:15.0'`
  - Conflicted with newer `org.jetbrains:annotations:23.0.0` pulled by updated dependencies
  - Caused duplicate class errors during build
  - Newer version from transitive dependencies is sufficient

### 9. ProGuard Rules Update
**File:** `android/proguard-rules.pro`
- **Added:** `-dontwarn android.media.LoudnessCodecController`
- **Added:** `-dontwarn android.media.LoudnessCodecController$OnLoudnessCodecUpdateListener`
  - Suppresses warnings for Android 14+ media classes referenced by Google Ads SDK
  - These classes are optional and not available on all Android versions
  - Required for successful release builds with R8/ProGuard minification
- **Added:** `-keep class com.lh9.feg1.firekidsgame.** { *; }`
- **Added:** `-keepclassmembers class com.lh9.feg1.firekidsgame.** { *; }`
- **Added:** `-keep class com.badlogic.gdx.** { *; }`
- **Added:** `-keepclassmembers class com.badlogic.gdx.** { *; }`
  - Prevents ProGuard from obfuscating or removing game classes
  - Prevents crashes in release builds caused by code optimization
  - LibGDX reflection requires classes to be kept

## Build Commands

### Using Build Scripts (Recommended)
```bash

./build_android.sh

# Or build individually:
```

### Manual Commands
To build the project:
```bash
# Debug build (for testing)
./gradlew android:assembleDebug

# Release build (signed, for production)
./gradlew android:assembleRelease

# Clean build
./gradlew clean android:assembleRelease
```

- Debug APK: `android/build/outputs/apk/debug/android-debug.apk`
